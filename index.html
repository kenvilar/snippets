this page is using this script https://cdn.jsdelivr.net/gh/kenvilar/snippets/index.min.js or this link https://kenvilar.github.io/snippets/index.min.js

So in your website, add:

<script defer src="https://cdn.jsdelivr.net/gh/kenvilar/snippets/index.min.js"></script>

or

<script defer src="https://kenvilar.github.io/snippets/index.min.js"></script>
Note:
jsDelivr caches the file, so it may still show the old version. You can force update it by purging the jsDelivr cache.
Open this URL in your browser: https://purge.jsdelivr.net/gh/kenvilar/snippets/index.min.js
It should return something like success/ok.
Then check CDN URL again: https://cdn.jsdelivr.net/gh/kenvilar/snippets/index.min.js
